# ka-chat-bot

A new and improved app template is now available. It uses a modern stack with Next.js and React, and includes a richer UI with support for reasoning, citations, and more. It also uses Neon with Lakebase integration to persist conversation history. The template works with all Databricks Agents and Foundation Models.

If you previously used the Marketplace template or the older chat app template, we strongly recommend moving to this new template:
https://github.com/databricks/app-templates/tree/main/e2e-chatbot-app-next
